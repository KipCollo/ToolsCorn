# PWA(Progressive Web Apps)

A progressive web application (PWA) is one that behaves more like a native application, which means it can continue working when there is no network connectivity, its code and content are cached so it can start immediately, and it can use features such as notifications. Progressive web application features are not specific to Angular.
The Angular team provides an NPM package that can be used to bring PWA features to Angular projects.

```sh
ng add @angular/pwa
```

The ng add command is used specifically to install packages, such as @angular/pwa, that have been designed to enhance or reconfigure an Angular project.
